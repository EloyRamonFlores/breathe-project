# Image Inventory — ToxinFree

Listado completo de todas las imágenes que necesita el sitio.
Cuando subas una imagen con el nombre exacto indicado a la carpeta correspondiente, avísame y actualizaré los JSON automáticamente.

**Carpeta raíz:** `public/images/`

---

## Cómo usar este archivo

1. Busca o genera la imagen (Unsplash, Wikimedia Commons, Midjourney, DALL-E, etc.)
2. Nómbrala exactamente como indica la columna **Nombre de archivo**
3. Colócala en la subcarpeta correspondiente dentro de `public/images/`
4. Márcala con `[x]` en este archivo cuando esté lista
5. Cuando quieras que actualice el código, dímelo

**Formato recomendado:** JPG, mínimo 1200px de ancho
**Proporción recomendada:** 16:9 para educativas/países, 1:1 para materiales

## Fuentes recomendadas

- **Wikimedia Commons** — fotos técnicas de materiales de construcción, licencia libre
- **Flickr (CC BY)** — fotografías industriales y de arquitectura históricas
- **Midjourney / DALL-E** — generación de imágenes técnicas específicas (ej. "close-up corrugated asbestos cement roof tiles weathered gray rust 1960s")
- **Unsplash** — solo para monumentos/países (categoría B), NO para materiales técnicos
- **Getty / Shutterstock** — recurso de pago, mayor variedad técnica
