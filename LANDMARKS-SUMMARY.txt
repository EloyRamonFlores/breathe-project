RESEARCH METHODOLOGY
====================

1. Global Recognition Test
   - Landmarks identifiable across demographics and cultures
   - Immediate visual recognition (show photo, can name landmark)
   
2. Photogenic Quality
   - Visually striking and distinctive
   - Good for hero imagery and card displays
   - Strong contrast and clear focal points

3. Architectural Preference
   - Prioritized built structures over natural features
   - Where nature is defining (falls, mountains), included
   
4. Cultural Significance
   - Represents national identity
   - Historic, religious, or symbolic importance

5. Unsplash Availability
   - High-quality, copyright-free images
   - Permanent URLs with reliable CDN
   - Free for commercial and non-commercial use

IMAGE SPECIFICATIONS
====================

- Source: Unsplash (https://unsplash.com)
- Quality: 1280px width (responsive parameter: ?w=1280)
- Format: JPEG, optimized for web
- Licensing: Free, no attribution required
- Coverage: All 37 countries have available images
- CDN: Unsplash global CDN ensures fast delivery

QUICK INTEGRATION EXAMPLE
=========================

// Hero section with landmark image
import landmarks from '@/data/landmarks.json';

export function CountryHero({ countrySlug }) {
  const landmark = landmarks.find(l => l.slug === countrySlug);
  
  return (
    <div className="relative w-full h-96 bg-cover"
         style={{ backgroundImage: `url(${landmark.unsplash_url})` }}>
      <div className="absolute inset-0 bg-black/30" />
      <h1 className="absolute bottom-8 left-8 text-white text-4xl font-bold">
        {landmark.name}
      </h1>
    </div>
  );
}
